# AppSec Portfolio

Bu repo: write-up, rapor, threat model, checklist ve CI/CD demo artefact'larini tek yerde toplar.

## Format
Her finding icin: Scope / Steps / Evidence (sansurlu) / Impact / Severity / Recommendation / Retest / Exec summary

## Klasorler
- writeups/: PortSwigger lab notlari
- reports/: Juice Shop ve diger raporlar
- threat-models/: threat model dokumanlari
- checklists/: code review checklist
- ci-cd-demo/: pipeline ve output ekranlari
- templates/: kopyala-yapistir sablonlar
