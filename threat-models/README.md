This folder contains lightweight threat models written as professional deliverables (system decomposition, DFD, trust
boundaries, prioritized threats, mitigations, and a validation plan).

## Index

## Conventions
- All content is written in English.
- Diagrams are provided using Mermaid where possible.
- Screenshots are used only for system mapping (not as "evidence" like lab write-ups).
